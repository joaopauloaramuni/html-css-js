Counter-Strike

Tecle " para abrir o debug. Depois digite os seguintes códigos:

Ajustar a gravidade sv_gravity e um número entre -999-999999
Auto-Reload +reload para habilitar -reload para desabilitar.
Movimentos mais rápidos cl_forwardspeed (# e um nº entre 0 e 999) para andar mais rápido para frente; cl_backwardspeed (# e um nº entre 0 e 999) para andar mais rápido para trás; cl_sidespeed (# e um nº entre 0 e 999) para andar mais rápido para os lados.
Escolha de cenário Changelevel (e o nome do cenário)
Escolha o tempo de detonação do C4 Mp_c4timer (# e um número entre -1000 e 1000
Acerte sempre sv_clienttrace 999999999. Todos os tiros vão sempre acertar. O default é 1.

Doom 3

Durante o jogo pressione ao mesmo tempo Ctrl, Alt e ~ para mostrar o console, depois entre com um dos seguintes códigos:

Dicas gerais
god - Fica invencível.
give all - Fica com todas as armas, energia cheia e armor cheio.
noclip - Para atravessar paredes e subir e descer em qualquer desnivel.
aviDemo - Grava a jogada num arquivo AVI.
benchmark - Faz um benchmark do jogo.
com_showfps 1 - Mostra os FPS do jogo.
com_showfps 0 - Oculta os FPS do jogo.
gfxinfo - Mostra informações da placa de vídeo.
give keys - Fica com todas as chaves.
g_nightmare 1 - Destrava o modo Nightmare.
g_nightmare 0 - Trava o modo Nightmare.
g_showplayershadow 1 - Mostra a sombra do jogador no modo single-player.
g_showplayershadow 0 - Oculta a sombra do jogador no modo single-player.
notarget - Fica invisível para alguns inimigos.
pm_thirdperson 1 - Visão em terceira pessoa.
pm_thirdperson 0 - Visão em primeira pessoa.
status - Mostra o status.
quit - Sai do jogo.
