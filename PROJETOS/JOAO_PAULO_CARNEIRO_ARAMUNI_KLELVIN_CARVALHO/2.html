Uso de tabelas em HTML

As tabelas constituem uma das formas de se controlar a largura de um texto dentro de uma página, evitando que a página sofra deformações visuais muito "radicais" quando vista em monitores de diferentes tamanhos.

Por exemplo:
<TABLE WIDTH=750><TR><TD>
...
... (sua página viria aqui)
...
</TD></TR></TABLE>

O artifício acima (colocar todo o conteúdo de sua página dentro de uma tabela de tamanho fixo que funciona, neste caso, como um QUADRO de 750 pixels de largura) contorna de certa forma este problema. Este tamanho é ideal para monitores de 14" e 800x600.
