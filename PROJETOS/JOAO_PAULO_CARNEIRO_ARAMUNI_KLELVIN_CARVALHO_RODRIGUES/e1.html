Uso de CSS

Podemos usar estilos de diversas maneiras, mas o que deve ficar muito claro é que:
  • Estilos foram feitos para dar um poder de alteração visual muito maior ao HTML;
  • Preferencialmente, deve-se utilizar o estilo em um arquivo externo ao código HTML
  • O número de formatações de estilo é muito grande; logo, é necessária uma fonte de consultas!
As vantagens do uso de estilos CSS estão listadas na tabela a seguir:
1 Pode-se alterar o visual de várias páginas alterando um único arquivo;
2 O controle visual dos elementos de uma uma página é muito maior;
3 Há efeitos que conseguimos apenas com o uso de CSS.
